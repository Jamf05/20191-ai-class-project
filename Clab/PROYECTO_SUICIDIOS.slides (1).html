MOTIVACIN:

Contenido:¶

Este conjunto de datos compilado se extrajo de otros cuatro conjuntos de datos vinculados por tiempo y lugar, y se cre para encontrar seales correlacionadas con el aumento de las tasas de suicidio entre diferentes cohortes a nivel mundial, en todo el espectro socioeconmico.

Columnas:¶

pas, ao, sexo, grupo de edad, conteo de suicidios, poblacin, tasa de suicidios, clave compuesta pas-ao, IDH por ao, gdp_for_year, gdp_per_capita, generacin (segn el promedio de agrupacin por edad).

Definiciones:¶

  • HDI (IDH): Indice de desarrollo humano. Es un indicador sinttico de los logros,medios obtenidos en las dimensiones fundamentales del desarrollo humano, a saber, tener una vida larga y saludable, adquirir conocimientos y disfrutar de un nivel de vida digno.
  • GDP (PIB): Producto interno bruto. En unidades de (Dlares estadounidenses actuales). Las fuentes incluyen: Banco Mundial, Naciones Unidas.

Objetivos :
¶

Regresion (La clases pertenecen a los reales):¶

  • Predecir el numero de suicidos (suicides_no) para unos determinados datos de entrada (country, year, sex, age, population?).
  • Precedir la probabilidad de que una persona se suicide (suicides_no(desconocido) / population) segun ciertas condiciones (Locacion especifica, condiciones de vida, ...)

    **Posibles complicaciones**: Solo se estaria determinando la probabilidad a partir de las condiciones de vida economicas del pais dejando de lado las personales, hasta que punto es posible realizar regresion? teniendo en cuenta la caractaristica **year**.

  • Determinar patrones de acuerdo a ciertas caracteristicas que presenten mayor influencia en las tazas de suicidios.

Zeros

Aquellos paises con numeros de suicidios suidices_no en 0 y aos antes y despues no muy lejanos presentan valores con cierta tendencia es muy probable que ese registro no fuera tomado y/o se haya perdido por lo tanto lo mejor seria resplazarlos con la media.

def zeros(condicion, umbral):
  # Da informacion sobre que paises poseen una mayor cantidad de registros con numeros de suicidios en 0

  #umbral: porcentaje [0-100]
  #country_N: nombres de paises
  #country_No: codigo de paises
  #N_zeros: cantitad de registros que NO tienen 0
  #N_total: total de registros
  #NOTA: La columna "country" ya debe haber sido convertida a codigos

    cod_pais = []
    nom_pais = []
    no_zeros = []
    no_registros = []
    porcentaje = []

    for i in country_codes.Numeric_code.values:
        country_N = country_codes.English_short_name_lower_case[country_codes.Numeric_code == i].values[0]
        country_No = country_codes.Numeric_code[country_codes.Numeric_code == i].values[0]
        N_zeros = np.count_nonzero(d.suicides_no[d.country == i])
        N_total = len(d.suicides_no[d.country == i])
        if (N_total != 0):
            P_final = 100 - float(N_zeros)/N_total*100
            if (P_final > umbral and condicion == "mayor"):  #Umbral de filtrado
                cod_pais.append(country_No)
                nom_pais.append(country_N)
                no_zeros.append(N_total - N_zeros)
                no_registros.append(N_total)
                porcentaje.append(P_final)
            if (P_final < umbral and condicion == "menor"): #Umbral de filtrado
                cod_pais.append(country_No)
                nom_pais.append(country_N)
                no_zeros.append(N_total - N_zeros)
                no_registros.append(N_total)
                porcentaje.append(P_final)

    zeros = {"cod_pais": cod_pais,
           "nom_pais": nom_pais,
           "no_zeros": no_zeros,
           "no_registros": no_registros,
           "porcentaje": porcentaje}
    zeros_df = pd.DataFrame(data=zeros, columns = ["cod_pais","nom_pais","no_zeros","no_registros","porcentaje"])
    return zeros_df
